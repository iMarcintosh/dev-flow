# DevFlow Logos

Place your logo files here:

## Files
- `devflow-logo-large.png` or `.svg` - Large logo for login page (recommended: 300-500px width)
- `devflow-logo-horizontal.png` or `.svg` - Horizontal logo for sidebar (recommended: 150-200px width)

## Formats
- SVG is preferred for scalability
- PNG with transparent background also works well

## Usage
- Large logo: Used in LoginPage, centered above login form
- Horizontal logo: Used in Sidebar, top-left corner

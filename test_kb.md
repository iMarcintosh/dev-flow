# DevFlow Knowledge Base

This is a test document about DevFlow features.

## Custom Agents
DevFlow supports creating custom AI agents with specific roles and capabilities.

## Knowledge Base Feature
Agents can have their own knowledge base by uploading documents. The system uses:
- Text extraction from PDFs and markdown
- Embedding generation with OpenAI
- Vector search with ChromaDB
- Semantic search for relevant information

## Tools Available
Custom agents can use various tools:
1. Board management (create/update tasks)
2. Web search (find information online)
3. Code execution (run Python, JavaScript, Bash)
4. Knowledge base search (search uploaded documents)

This enables powerful workflows!
